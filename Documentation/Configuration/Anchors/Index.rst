.. _commentAnchorPrefix:

commentAnchorPrefix
"""""""""""""""""""
.. container:: table-row

   Property
      commentAnchorPrefix
   Data type
      string
   Default
      comment
   Description
      Every comment gets an own anchor which contains the uid of the comment. In the default case the anchor will
      look something like that: #comment1337.
